Editing on Android

Published on: Dec 20, 2025

I can’t access my laptop for a month, so I decided to make a HTML editing setup on my android device, so I can update my website regularly. I have made the the sample templates already in VS Codium, so I just have to edit them in the code editor and my page is up and running.

I am using Acode editor for editing the html of my website. Acode is a Free and Open Source Software like most of the software I use. I have installed it last week and it felt like a complete IDE, meaning, I can’t differentiate between the features of VS Codium and Acode. You can also edit the html in notepad of the computer, not sure about the possibility in android, but I find it easier to edit in IDE. Auto-completion feature in IDE makes my job as a webmaster easier.

I downloaded it from F-droid and it is also available on Google Play Store. There is an option to install different extensions on Acode like VS Codium, I installed HTML5_boilerplate plugin to check if extensions work or not, and it works as intended. It also has a terminal but I didn’t used it, because it gave me some error when I tried to install it. I am sure after some tweaking I can use it but, I don’t need the terminal and I don’t want to spend my time on troubleshooting.

One thing I dont like is, it shows ads when you are connected to the internet apart from that I don’t have any problem with it for now. So, for a month this will be my setup, and I’ll make a complete one month experience post once I get back my laptop. Till then all my posts will be written and edited in my android device.
